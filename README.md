## How to setup locally

### Run On Local Machine

```bash
git clone https://github.com/1SAURABHKUMAR1/Coder-Social.git
```

### Setup

Go to the project directory

```bash
  cd Coder-Social
```

Install dependencies for Frontend

```bash
  cd Frontend
  npm install
```

Install dependencies for Backend

```bash
  cd Backend
  npm install
```

Start the server

```bash
  npm run start
```
